## Row — Контейнер для колонок

Компонент `Row` используется для организации сетки в макете. Он группирует `Col` элементы и управляет их выравниванием и отступами.

### Основные возможности:

- **Гибкое управление отступами между колонками (`gutter`)**
- **Выравнивание колонок по горизонтали и вертикали (`justify`, `align`)**
- **Возможность включения и отключения переноса (`wrap`)**
