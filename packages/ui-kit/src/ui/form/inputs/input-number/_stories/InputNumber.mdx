InputNumber - это расширенный компонент для ввода числовых значений. Поддерживает валюты, десятичные числа, floating labels и различные форматы отображения.

## Основные возможности:

- **Валютные форматы**: Поддержка RUB, USD, EUR с настройкой позиции символа
- **Floating Labels**: Анимированные лейблы как в Material Design
- **Форматирование**: Настройка разделителей тысяч и десятичных знаков
- **Контролы**: Возможность скрытия стрелок управления
- **Размеры**: Small, Middle, Large
