## Steps — Навигационный компонент

Компонент `Steps` используется для отображения пошагового прогресса в процессах.

### Основные возможности:

- **Горизонтальная и вертикальная ориентация**
- **Разные статусы (wait, process, finish, error)**
- **Возможность отображения прогресса в виде точек**
